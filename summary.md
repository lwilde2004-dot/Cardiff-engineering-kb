## 1. CLAUDE.md (Root Prompt)

Full content is at `CLAUDE.md`. Key rules:

- **Purpose**: Cardiff University EN1213/EN1214 engineering knowledge base. Lewis drops raw lecture files; Claude builds and maintains the wiki.
- **Intake flow**: Rename → `raw/` → `convert.sh` → tell Claude to process it.
- **When processing a lecture**, Claude must:
  1. Read the converted `.md`
  2. Update or create the relevant wiki article(s)
  3. Ensure the `.md` has YAML frontmatter
  4. Add the lecture to `_modules/MODULE-CODE.md`
  5. Update `wiki/INDEX.md` if a new article was created
  6. Add `## Key Diagrams` section (2–4 images) before `## Links` using full vault-root paths

## 3. Naming Convention

`MODULE CODE - Lecture Title - Lecturer Surname.ext`

Examples:
- `EN1213 - Hydrostatics and Submerged Surfaces - Mashruk.pptx`
- `EN1213 - Solid Mechanics Introduction - Pearson.pdf`
- `EN1213 - Atomic Structure and Bonding - Williams.pptx`

EPUBs only need `MODULE CODE - Title.epub` (no lecturer).

## 6. Practice Questions Format

File location: `EN1213/{subject}/practice-questions/{topic}.md`

File header:
```markdown
---
tags: [statics, mechanics, practice-questions]
---

# Statics — Practice Questions

*Questions collated from study sessions. Format: Given → Find → Solution → Answer.*

---
```

Each question:
```markdown
## Q[N] — [Topic / Sub-concept] — [Easy/Medium/Hard]

**Given:** Describe the scenario with all known values and units.

**Find:** What must be calculated.

---

**Solution:**

**Step 1 — [Label]:** Explanation.
$$\text{LaTeX equation}$$

**Step 2 — [Label]:** ...

**Answer:** Numerical result with units
```

Number questions sequentially (check the last Q number before adding). Always show full working. Always include units.

## 11. Key Scripts

### convert.sh (wrapper)
```bash
bash "_system/convert.sh" "path/to/file.pptx"
```
Calls `convert_to_md.py` with auto-detected module/topic/lecturer from filename.

### intake.sh (smart intake)
```bash
bash "_system/intake.sh" "path/to/file.pptx"
```
Prompts for topic interactively, renames to convention, moves to correct `raw/` folder, converts.

### check.sh (protocol check)
```bash
bash "_system/check.sh"
```
Checks: naming convention, .md counterparts, frontmatter, module indexes, subject folders, web clips inbox, web clipper templates, stranded files, _inbox backlog.
