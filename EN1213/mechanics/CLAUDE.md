# Knowledge Base Schema — Mechanics (Statics & Dynamics)

## What This Is
A personal knowledge base for Cardiff University engineering mechanics modules covering statics and dynamics.

## How It's Organized
- raw/ contains unprocessed source material (lecture slides, notes, problem sheets). Never modify these files.
- wiki/ contains the organised wiki. AI maintains this entirely.
- outputs/ contains generated summaries, answers, and revision notes.

## Wiki Rules
- Every topic gets its own .md file in wiki/
- Every wiki file starts with a one-paragraph summary
- Link related topics using [[topic-name]] format
- Maintain an INDEX.md in wiki/ listing every topic with a one-line description
- When new raw sources are added, update the relevant wiki articles

## My Interests
- Statics: equilibrium, free body diagrams, trusses, moments
- Dynamics: kinematics, Newton's laws, energy and momentum methods
- Rigid body motion
- Problem-solving approaches for exam questions
