# Knowledge Base Schema — Thermodynamics & Fluid Mechanics

## What This Is
A personal knowledge base for Cardiff University engineering modules covering thermodynamics and fluid mechanics (EN1103, EN1213, EN2103).

## How It's Organized
- raw/ contains unprocessed source material (lecture slides, notes, lab data). Never modify these files.
- wiki/ contains the organised wiki. AI maintains this entirely.
- outputs/ contains generated summaries, answers, and revision notes.

## Wiki Rules
- Every topic gets its own .md file in wiki/
- Every wiki file starts with a one-paragraph summary
- Link related topics using [[topic-name]] format
- Maintain an INDEX.md in wiki/ listing every topic with a one-line description
- When new raw sources are added, update the relevant wiki articles

## My Interests
- Ideal gases and thermodynamic cycles (steam, Rankine)
- First and second laws of thermodynamics, entropy
- Fluid statics and dynamics (Bernoulli, momentum, pipe flow)
- Dimensional analysis
- Lab work and experimental data analysis
