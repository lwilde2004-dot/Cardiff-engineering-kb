# Knowledge Base Schema — Manufacturing & Materials

## What This Is
A personal knowledge base for Cardiff University engineering modules covering manufacturing processes and materials science (EN1213 and related modules).

## How It's Organised
- raw/ contains unprocessed source material (lecture slides, lab reports, notes). Never modify these files.
- wiki/ contains the organised wiki. AI maintains this entirely.
- outputs/ contains generated summaries, revision notes, and formula sheets.

## Subfolders
- raw/materials-science/ — material structure, properties, phase diagrams, material selection
- raw/manufacturing-processes/ — machining, casting, forming, welding, 3D printing, injection moulding
- raw/materials-testing/ — tensile testing, hardness, fatigue, creep, impact

## Wiki Rules
- Every topic gets its own .md file in wiki/
- Every wiki file starts with a one-paragraph summary
- Link related topics using [[topic-name]] format
- Maintain an INDEX.md in wiki/ listing every topic with a one-line description
- When new raw sources are added, update the relevant wiki articles

## My Interests
- Material properties (mechanical, thermal, electrical)
- Manufacturing process selection and comparison
- Lab work: tensile testing, creep, hardness measurement
- Sustainable manufacturing and material selection
