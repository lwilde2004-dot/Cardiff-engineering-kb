# Knowledge Base Schema — Coding

## What This Is
A personal knowledge base for programming and computational methods relevant to Cardiff University engineering.

## How It's Organized
- raw/ contains unprocessed source material (lecture notes, code snippets, tutorials). Never modify these files.
- wiki/ contains the organised wiki. AI maintains this entirely.
- outputs/ contains generated summaries, example programs, and cheat sheets.

## Wiki Rules
- Every topic gets its own .md file in wiki/
- Every wiki file starts with a one-paragraph summary
- Link related topics using [[topic-name]] format
- Maintain an INDEX.md in wiki/ listing every topic with a one-line description
- When new raw sources are added, update the relevant wiki articles

## My Interests

- Object-oriented programming concepts
- Computational methods for engineering (numerical methods, data processing)
- Build and compile workflow (clang on macOS)
