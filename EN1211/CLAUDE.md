# Knowledge Base Schema — EN1211 Engineering Maths and Computation

## What This Is
A personal knowledge base for Cardiff University EN1211 Engineering Mathematics. Source material is lecture notes; Claude maintains the wiki from them.

## Vault Structure

```
EN1211/
  raw/topic/          ← Source PDFs and converted .md. Never modify.
  wiki/topic/         ← AI-maintained articles, one .md per topic
  outputs/            ← Revision notes, formula sheets, worked examples
  flashcards/         ← Spaced-repetition flashcards (tags: [flashcards])
```

## Wiki Rules
- Every topic has one `.md` file in its `wiki/` folder plus an `INDEX.md`
- Each article starts with a one-paragraph summary
- Link related topics using `[[topic-name]]` format
- Maintain `wiki/INDEX.md` listing every article
