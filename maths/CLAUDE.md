# Knowledge Base Schema — Mathematics

## What This Is
A personal knowledge base for Cardiff University engineering mathematics.

## How It's Organized
- raw/ contains unprocessed source material (lecture notes, problem sheets, worked examples). Never modify these files.
- wiki/ contains the organised wiki. AI maintains this entirely.
- outputs/ contains generated summaries, formula sheets, and worked solutions.

## Wiki Rules
- Every topic gets its own .md file in wiki/
- Every wiki file starts with a one-paragraph summary
- Link related topics using [[topic-name]] format
- Maintain an INDEX.md in wiki/ listing every topic with a one-line description
- When new raw sources are added, update the relevant wiki articles

## My Interests

- Linear algebra (matrices, vectors)
